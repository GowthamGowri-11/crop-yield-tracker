Backend Server

http://localhost:5000

Click the "Test" button on any endpoint to see the response

GET /api/crop-yields
Get all crop yields with advanced filtering. Supports pagination, sorting, search, and multiple filters.
Filters: cropType, farmerName, location, minWeight, maxWeight, startDate, endDate, search
GET /api/crop-yields/stats
Get comprehensive yield statistics including crop-wise breakdown and overall metrics.
Filters: startDate, endDate, cropType, location
GET /api/farmers/stats
Get farmer performance statistics with total yields, averages, and crop diversity.
GET /api/crop-yields/trends?year=2024
Get monthly yield trends for a specific year with crop-wise breakdown.
GET /api/locations/stats
Get location-based statistics showing performance by farm location.
GET /api/crop-yields/compare?crops=Rice,Wheat,Corn
Compare multiple crops side-by-side with detailed metrics.
GET /api/top-performers?type=farmers&limit=5
Get top performers by farmers, locations, or crops. Supports time period filtering.
Parameters: type (farmers/locations/crops), limit, period (week/month/year/all)
